# Colors

Color will automatically match color-scheme, and if light-dark is not supported, it will fall back to dark value.

Set color-scheme for the document root or element to change the system default value.

```html
<html style="color-scheme: light">
</html>
```

```html
<html style="color-scheme: dark">
</html>
```
