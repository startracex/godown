# Colors

Godown using [light-dark](https://developer.mozilla.org/en-US/docs/Web/CSS/color_value/light-dark) function to support both light and dark mode.

All major browsers released after Baseline 2024 support this feature. However, in cases where the feature is not supported, it will fall back to the "dark" value.

Set color-scheme for the document root or element to override scheme.

```html
<html style="color-scheme: light"></html>
```

```html
<godown-alert style="color-scheme: dark"></godown-alert>
```
