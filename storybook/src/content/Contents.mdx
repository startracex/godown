# Contents

Set the `display` property of the host element to `contents`.

## Examples

### Normal

<godown-button>Normal</godown-button>

```html
<godown-button>Normal</godown-button>
```

### Contents

<godown-button contents>Contents</godown-button>

```html
<godown-button contents>Contents</godown-button>
```

## How it works

In shadowRoot, the element with `part="root"` or first element child is considered root.

The host element display will be content.

All styles of the root element inherit, but display is [revert](https://developer.mozilla.org/en-US/docs/Web/CSS/revert).

Apply the following style:

```css
:host([contents]) {
  display: contents;
}

:where(:host([contents]) [part="root"]) {
  all: inherit;
  display: revert;
}
```

`getBoundingClientRect` and `getClientRects` will redirect to the root element.

```js
const button = document.querySelector("godown-button[contents]");
button.getBoundingClientRect(); // root element
HTMLElement.prototype.getBoundingClientRect.call(button); // host element
```
